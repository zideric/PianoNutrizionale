# Directory Overview

This directory contains a weekly nutritional plan for two people, Eric and Chiara. The plan is available in two formats: an interactive HTML page and a plain text file.

# Key Files

*   `index.html`: An interactive HTML page that displays the weekly nutritional plan. It includes daily meal plans, a weekly table, and a list of recipes. The page is interactive, allowing the user to switch between different views (daily plan, weekly table, recipes) and select different days of the week.
*   `menu_settimanale_dettagliato.txt`: A plain text file that contains the same information as the `index.html` file.

# Usage

The `index.html` file can be opened in a web browser to view the nutritional plan in an interactive format. The `menu_settimanale_dettagliato.txt` file can be used for a quick reference or for printing the plan.
